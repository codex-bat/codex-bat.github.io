StreamerUtils

server‑side Fabric mod — chat & Twitch streaming integration

StreamerUtils is a server‑side Fabric mod that supercharges chat and Twitch integration for streamers and their communities. It adds custom nameplate icons, coloured usernames, live status, follow alerts, and interactive chat commands – all configurable per player and saved across sessions.

Whether you run a streamer‑friendly server or just want a unique nametag, this mod gives you (and your viewers) a more interactive experience. Viewers can use !highlight to make you glow, !firework to launch a rocket, and you get sound alerts for every new follower – all without any client mods (though a companion resource pack makes icons actually visible).

I plan to add more platform icons (Discord or whatever other random platforms you are interested in), but the current version is fully functional and stable already.

If you want to inspect the code or verify its safety, the source is available on GitHub. The mod currently packages Twitch4J with it too, which is actually the main reason it's around 17k KBs. I plan on adding it as a dependency instead, later.

Supports Minecraft 1.21.11 (other versions may work with updated mappings). Future updates will target 26.1/2 (or perhaps some requested versions, if people are interested) and add more features.

If you have any questions or suggestions, hit me up on Discord (as usual)!
